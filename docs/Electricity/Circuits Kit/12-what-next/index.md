If you made it this far, you have already done a lot.

You have worked with:

- LEDs
- buttons
- resistors
- breadboards
- GPIO pins
- code
- PWM
- timing
- animation

That is a real foundation.

## What to do next

A good next step is to start changing the projects on purpose.

Try things like:

- changing the timing
- changing the button behavior
- mixing two older projects together
- adding your own game rules
- making the lights react in a new pattern

## Project ideas

Try one of these next:

- Remix [3: Two Buttons OR](../3-two-buttons-or/index.md) with different button rules.
- Extend [4: Two Buttons AND](../4-two-buttons-and/index.md) by adding a third button.
- Turn [7: LED Chase](../7-led-chase/index.md) into your own custom light pattern.
- Add more LEDs to [7: LED Chase](../7-led-chase/index.md)
- Upgrade [9: Traffic Light](../9-traffic-light/index.md) with a button-controlled crosswalk idea.
- Revisit [Electricity](../../index.md) if you want to better understand the circuit ideas behind the projects.

## A good challenge

Pick one earlier project and change just one thing at a time.

That is one of the best ways to learn, because you can tell what each change actually did.

## Keep building

You do not need a giant project to keep learning. Small changes, careful testing, and good questions go a long way.
